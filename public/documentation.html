API Documentation

Movies

Request URL Response
GET /movies List of ALL movies
GET /movies/:title Data about a single movie by title
GET /movies/genre/:genreName Data about a genre by name/title (e.g., “Thriller”)
GET /movies/directors/:directorName Data about a director (bio, birth year, death year) by name

Users

Request URL Response
POST /users New user registration
PUT /users/:id Update user info (username)
POST /users/:id/:movieTitle Add a movie to user's list of favorites
DELETE /users/:id/:movieTitle Remove a movie from user's list of favorites
DELETE /users/:id Deregister an existing user
